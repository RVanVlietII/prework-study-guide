HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.
  • The base defines the Base URL for the webpage.
  • The char set element sets the character encoding.
  • The http equiv set a HTTP header for content.
  • The viewport sets the default view of the webpage.
  • The opening tags with angle brackets and closing tags with / with angle bracket allow us to start and stop a portion of code.
  • The img sets the data for the element of the image.
  • The src sets the location of the img.
  • The br creates a line break between two blocks of content.
  • The rel is relationship reference and href is external URL reference.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A style defines the CSS style for the website.
  • CSS rules: declarations must have property and associated value.
  • We use the text-align command to define how the text will align in css.
  • The img is assigned property values correlated with img in HTML.
  • The css must be linked with rel and href.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A adds all changes added to main branch
  • git commit -m "with comment"
  • git pull pushes data to origin branch
  • git checkout main: swithces back to main branch
  • git push: uploads local content into remote repo
  • git commit saves changes on local repository

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An array is a variable that holds groups of values with brackets and commas.
  • The console log is for displaying the variable listed.
  • We use a for loop to console.log many variables and evaluate the array automatically.
  • The "for" loop contains a starting point, condition, and final statement or loop pattern.
  • Conidtional statements provide decision making for the code to execute.
  • Truthy statements utilize "===" for boolean phrases as T/F.
  • A function must be named to execute the code of instruction that proceeds it in the parantheses.
  • A function makes a for loop and if, then statement easier to execute repeatededly.
